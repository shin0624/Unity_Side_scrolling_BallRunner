###### **5. 기능**
----
**횡스크롤 형태의 공 튀기기 게임**


**게임이 시작하면 일정 속도로 공이 이동하며, 점프로 발판에서 구멍을 넘어 진행시간이 증가할 수록 속도와 구멍 넓이, 발판 높이가 증가**


점프 : 마우스 왼쪽 버튼


화면 좌측 아래에 점프 횟수 카운트를 표시

구멍에 빠지면 카운트가 초기화되며, 시작 지점으로 돌아온다.
